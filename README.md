# Analyse RH — Turnover et Parité Salariale

##  Contexte
Analyse complète des données RH d'une entreprise
de 20 employés pour identifier les causes du turnover
et analyser la parité salariale par département.

## Problématique
- Pourquoi le taux de départ est-il de 30% ?
- Quels départements sont les plus touchés ?
- Existe-t-il des écarts salariaux entre H/F ?

## Insights clés
- Le département RH a un taux de départ de 75% ⚠️
- RH est aussi le département le moins bien payé (30 375€)
- L'écart salarial global favorise les femmes (+1 350€)
  mais s'explique par la répartition des postes
- IT est le seul département où les hommes
  gagnent plus à poste équivalent (+5 000€)

##  Recommandations
1. Revoir la politique salariale RH en priorité
2. Investiguer l'écart salarial IT Analyste
3. Mettre en place un suivi mensuel du turnover
